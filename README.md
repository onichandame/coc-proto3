# coc-proto3

coc.nvim extension for protobuf. Features include:

- [x] go-to definition
- [x] compile proto

## Install

`:CocInstall coc-proto3`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-proto3-keymap)`

## Lists

`:CocList demo_list`
